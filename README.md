# 📊 Sales & Revenue Analysis Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze sales and revenue data using the Sample Superstore dataset. It provides key business insights through KPIs, charts, and interactive slicers.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold
- Total Orders
- Monthly Sales Trend
- Top 10 Products
- Sales by Region
- Category-wise Sales
- Interactive Slicers

## Tools Used
- Power BI
- Power Query
- Microsoft Excel

## Dataset
Sample Superstore Dataset
